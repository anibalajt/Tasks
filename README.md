# Tasks

## Installation

If you want to test the app on a simulator running locally:

`git clone https://github.com/anibalajt/Tasks.git`

```bash
$ cd Tasks

$ yarn install or npm install

$ cd ios

$ pod install

$ cd ..

$ react-native run-ios
```
